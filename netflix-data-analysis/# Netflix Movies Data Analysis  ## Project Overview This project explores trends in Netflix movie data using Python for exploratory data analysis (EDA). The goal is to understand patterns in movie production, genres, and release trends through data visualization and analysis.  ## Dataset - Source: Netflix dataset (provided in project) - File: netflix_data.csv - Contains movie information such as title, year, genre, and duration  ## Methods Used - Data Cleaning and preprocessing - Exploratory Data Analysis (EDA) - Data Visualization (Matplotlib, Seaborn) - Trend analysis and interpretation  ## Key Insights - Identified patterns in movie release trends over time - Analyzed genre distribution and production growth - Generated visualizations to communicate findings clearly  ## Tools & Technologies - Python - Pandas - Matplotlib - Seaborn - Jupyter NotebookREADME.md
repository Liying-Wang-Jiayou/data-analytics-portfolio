# Netflix Movies Data Analysis

## Project Overview
This project explores trends in Netflix movie data using Python for exploratory data analysis (EDA). The goal is to understand patterns in movie production, genres, and release trends through data visualization and analysis.

## Dataset
- Source: Netflix dataset (provided in project)
- File: netflix_data.csv
- Contains movie information such as title, year, genre, and duration

## Methods Used
- Data Cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization (Matplotlib, Seaborn)
- Trend analysis and interpretation

## Key Insights
- Identified patterns in movie release trends over time
- Analyzed genre distribution and production growth
- Generated visualizations to communicate findings clearly

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
